# Chat Box
```Chat-Box``` is an app that allows users to participate in group messages with their friends.
Users can take pictures, send pictures from their libraries as well as share their location, 
in addition to standard messaging

## Technologies Used
- React Native
- Expo
- Firebase

## Dev Dependancies
- Babel

## Useage
Users can choose their display name upon signing in, and choose the color of their message bubbles.
Sent messages are saved to Firestore so they persist after users log off and React Native Async Storage,
letting users access their messages even when they're offline.  Users can choose to send 
photos or their location by tapping the "Add Media" button on the left of the chat bar.
